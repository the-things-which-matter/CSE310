# Overview

This project is a Student Registration Web Application developed using Node.js and Express. The application allows users to register multiple students for the CS310 course and view all registered students through dynamically generated web pages.



This project also helped me practice organizing web applications using multiple pages, external styling files, reusable routes, and server-side programming concepts.

The software allows users to:
- Register one or multiple students
- View all registered students
- Navigate through multiple web pages
- Interact with dynamically generated content

# Web Pages

The application contains multiple web pages:

- Home Page  
  Allows users to register one or multiple students for the CSE243 course.

- Students Page  
  Displays all registered students dynamically.

- About Page  
  Provides information about the project and what the app does
